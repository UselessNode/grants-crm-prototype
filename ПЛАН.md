# План работы над проектом CRM для приема заявок на гранты

## Описание проекта
Я студент. Меня пригласил преподаватель поработать над прототипом CRM странички для приёма заявок на подачу грантов. Если организация выделит деньги, то я (и команда) будем работать над этим проектом. Пока это учебный проект.
