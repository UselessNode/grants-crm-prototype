# 💼 Grants CRM — Прототип CRM системы для заявок на гранты

## 🎯 О проекте

**Grants CRM** — это прототип системы для управления заявками на гранты.  

---

## Быстрый и не очень старт

### Локальный запуск (для разработчиков)

```bash
# Установка зависимостей
npm install

# Запуск режима разработки
npm run dev

# Сборка для продакшена
npm run build

# Сброс базы данных
npm run db:reset

# Миграции
npm run migrate

# Тестовые данные
npm run seed
```

---

## Тестовые учётные данные

В тестовом режиме для входа в систему используйте:

| Email | Пароль | Роль |
|-------|--------|------|
| `admin@example.com` | `test123` | Администратор |
| `user@example.com` | `test123` | Пользователь |
| `test@example.com` | `test123` | Пользователь |

Или зарегистрируйте нового пользователя на странице `/register`.

---

## Функционал прототипа

### Реализовано

- **Авторизация и регистрация**
  - Вход по email и паролю
  - Регистрация нового пользователя
  - Разделение на роли (администратор / пользователь)
  - Сессионные токены (7 дней)
  - Предупреждение о завершении сессии

- **Управление заявками**
  - Просмотр списка заявок с фильтрацией
  - Создание новой заявки
  - Редактирование существующей заявки
  - Удаление заявки
  - Детальный просмотр заявки

- **Фильтрация и поиск**
  - Поиск по названию и описанию
  - Фильтр по направлению
  - Фильтр по статусу
  - Пагинация результатов

- **Интерфейс**
  - Адаптивный дизайн
  - Компонентная архитектура
  - Централизованная система иконок
  - Стилизация через Tailwind CSS

## Моковые данные

В демонстрационном режиме используются следующие данные:

### Пользователи
- 2 предопределённых пользователя (admin, user)
- Возможность регистрации новых

### Заявки
- 5 предопределённых заявок в разных статусах
- Направления: Образование, Культура, Экология, Спорт, Наука, Волонтёрство
- Статусы: Черновик, Подана, На рассмотрении, Одобрена, Отклонена

## 🔧 Конфигурация

### Переключение режимов

В файле `frontend/src/utils/config.ts`:

```typescript
export const CONFIG = {
  MOCK_MODE: true,  // true = демо режим, false = реальный бэкенд
  API_URL: 'http://localhost:3001/api',
  MOCK_DELAY: 500,  // Задержка для имитации сети (мс)
};
```

## Для разработчиков

### API Endpoints (для будущей интеграции)

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход |
| GET | `/api/auth/verify` | Проверка токена |
| GET | `/api/auth/me` | Профиль пользователя |
| GET | `/api/applications` | Список заявок |
| POST | `/api/applications` | Создание заявки |
| PUT | `/api/applications/:id` | Обновление заявки |
| DELETE | `/api/applications/:id` | Удаление заявки |
