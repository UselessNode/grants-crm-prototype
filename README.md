# 💼 Grants CRM — CRM система для управления заявками на гранты

[![Last Commit](https://img.shields.io/github/last-commit/UselessNode/grants-crm-prototype/main?label=Last%20Commit&color=blue)](https://github.com/UselessNode/grants-crm-prototype)
[![Status](https://img.shields.io/badge/status-development-yellow)]()

**Grants CRM** — это система для управления заявками на гранты с полным циклом: от создания черновика до экспертизы и финального решения.

📅 **Актуализировано:** 23 марта 2026 г.

---

## 🚀 Быстрый старт

### Локальный запуск (для разработчиков)

```bash
# Установка зависимостей
npm install

# Запуск режима разработки (frontend + backend)
npm run dev

# Сборка для продакшена
npm run build

# Сброс базы данных
npm run db:reset

# Миграции БД
npm run migrate

# Тестовые данные (seed)
npm run seed
```

## ✅ Функционал

### Реализовано

#### 🔐 Авторизация и безопасность
- [x] Вход по email и паролю
- [x] Регистрация нового пользователя
- [x] Разделение на роли (администратор / пользователь)
- [x] Сессионные токены (7 дней)
- [x] Предупреждение о завершении сессии

#### 📝 Управление заявками
- [x] Создание заявки (статус «Черновик» по умолчанию)
- [x] Редактирование черновиков и отклонённых заявок
- [x] Подача заявки вручную (с проверкой обязательных полей)
- [x] Просмотр списка заявок в формате компактных карточек
- [x] Детальный просмотр заявки с сайдбаром экспертов
- [x] Удаление заявки (только черновики и отклонённые)

#### 🔍 Поиск и фильтрация
- [x] Поиск по названию и владельцу
- [x] Фильтр по статусу
- [x] Фильтр по направлению
- [x] Фильтр по эксперту
- [x] Массовый выбор заявок

#### 👥 Экспертиза заявок
- [x] Таблица экспертов (добавление, просмотр)
- [x] Назначение 2 экспертов на заявку
- [x] Массовое назначение экспертов из списка заявок
- [x] Вердикты экспертов (одобрено/отклонено)
- [x] Комментарии экспертов
- [x] Статусы заявок:
  - **Черновик** → редактирование доступно
  - **Подана** → пользователь не может редактировать
  - **На рассмотрении** → эксперты проверяют
  - **Одобрена** → можно редактировать
  - **Отклонена** → можно редактировать

#### 🎨 Интерфейс
- [x] Адаптивный дизайн
- [x] Компонентная архитектура
- [x] Стилизация через Tailwind CSS

## 🏗 Архитектура

### Технологический стек

| Компонент | Технология |
|-----------|------------|
| Frontend | React 18 + TypeScript |
| Сборка | Vite |
| Стили | Tailwind CSS |
| Роутинг | React Router v6 |
| Состояние | Zustand |
| HTTP-клиент | Axios |
| Backend | Node.js + Express |
| База данных | PostgreSQL |
| ORM | pg |

### Структура проекта

```
src/
├── frontend/              # Клиентская часть
│   └── src/
│       ├── components/    # UI компоненты
│       │   ├── AdminPanel/
│       │   ├── ApplicationForm/
│       │   └── ui/
│       ├── pages/         # Страницы
│       ├── hooks/         # React хуки
│       ├── store/         # Состояние (Zustand)
│       ├── utils/         # Утилиты и API
│       └── styles/        # Глобальные стили
├── backend/               # Серверная часть
│   └── src/
│       ├── controllers/   # Контроллеры
│       ├── models/        # Модели данных
│       ├── routes/        # Маршруты API
│       ├── middleware/    # Middleware (auth)
│       └── config/        # Конфигурация
└── scripts/               # Скрипты
    ├── migrations/        # Миграции БД
    └── seed_data.sql      # Тестовые данные
```

|------|----------|

## 🧪 Для разработчиков

### API Endpoints

#### Авторизация
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация |
| POST | `/api/auth/login` | Вход |
| GET | `/api/auth/verify` | Проверка токена |
| GET | `/api/auth/me` | Профиль пользователя |

#### Заявки
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/applications` | Список заявок |
| GET | `/api/applications/:id` | Заявка по ID |
| POST | `/api/applications` | Создание заявки |
| PUT | `/api/applications/:id` | Обновление заявки |
| DELETE | `/api/applications/:id` | Удаление заявки |
| POST | `/api/applications/:id/submit` | Подать заявку |
| POST | `/api/applications/:id/verdict` | Вердикт эксперта |

#### Админ-панель
| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/admin/stats` | Статистика |
| GET | `/api/admin/users` | Пользователи |
| GET | `/api/admin/applications` | Все заявки |
| GET | `/api/admin/experts` | Эксперты |
| POST | `/api/admin/experts` | Добавить эксперта |
| PUT | `/api/admin/applications/:id/experts` | Назначить экспертов |
| GET | `/api/admin/directions` | Направления |
| GET | `/api/admin/tenders` | Тендеры |
