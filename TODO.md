# План работ по приоритетам

## 🔴 Приоритет 1 (Критично — сделать в первую очередь)

- [ ] **Эксперты**. Новая таблица. Эксперты проверяют заявки статуса "**подана**". Заявки меняются на статус на "**рассмотрении**" и по результатам проверки эксперта меняется на другие статусы: "**одобрена**", "**отклонена**". Нужно **два эксперта на одну заявку** и наличие четкой кнопки "окей".
- [ ] **Статусы заявок** — реализовать логику переходов:
  1. **Черновик**. По умолчанию. Редактировать может и пользователь и админ.
  2. **Подана** (Пользователь не может редактировать, админ может редактировать и отправляет на рассмотрение)
  3. **На рассмотрении** (Никто не может редактировать, два эксперта рассматривают заявку)
  4. **Одобрено** (эксперт выдал вердикт. Пользователь и админ могут редактировать. Добавляется комментарий от экспертов что исправить)
  5. **Отклонено** (эксперт выдал вердикт. Вообще ничего нельзя сделать. Никто не может редактировать)
- [ ] **Подача заявки вручную** — только в окне просмотра, с подтверждением
- [ ] После создания заявки заявка должна быть в статусе **черновик**
